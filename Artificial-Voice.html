Application which helps deaf,blind,dumb person to communicate with normal sight person and vice versa.

Brief:

FAR:

It consists of two interfaces:

  • Chatting interface for a blind-deaf person.
  • Chatting interface for a normal person.

A deaf-blind person can send a message by tapping the screen of the app:

  • Short tap is dot ( . )
  • Long tap is dash ( - )

The Morse code written by the blind-deaf person is converted to plain English while sending the message to a normal person. Incoming messages from a Normal person to a deaf-blind person are converted into vibrations of dot & dash for the blind and deaf person.

NEAR:

This feature is used when both the blind-deaf person and normal person are at the same place and want to communicate with each other.

When a normal person speaks, it is converted into vibrations via Morse code in real-time for the blind-deaf person to understand the message.

The blind-deaf person can give a reply in real-time by tapping the screen of the app, and it is converted into speech.

Built With

microsoft cognitive services, GPT 3, Express, Node js, HTML,CSS,Javascript, microsoft Azure, Flutter
