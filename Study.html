Study

S

A web app that simplifies the process of making class notes for students.

Brief:

Our app converts speech into transcripts and finds relevant YouTube videos for students. The pandemic has popularized online classes, yet teachers struggle to maintain student engagement with online concepts. Our solution leverages YouTube and the Symbol.AI API to clarify classroom concepts, making learning more accessible and enjoyable.

YouTube is not merely a platform for sharing and uploading videos. It is a valuable educational resource that teachers and educators can exploit to maximize student benefits. By introducing fun elements into lessons through YouTube, we meet students' interests and enhance their learning experience. Numerous academic institutions already utilize YouTube to distribute course materials, making it an indispensable tool for modern education.

Our project features two interfaces: one for teachers and another for students. The process begins with teachers recording audio of their lessons using the Symbl.AI Conversational API (SPEECH TO TEXT), which is then converted into transcripts. After class, teachers can share comprehensive notes with their students, ensuring no detail is missed.

In the student portal, the text shared by teachers is accessible. Students can then use the conversational API (GET TOPICS) to extract keywords from these notes. Based on these keywords, our app uses the YouTube API to suggest relevant videos, enabling students to explore topics in a fun and interactive manner. This method not only helps students master classroom topics from diverse internet sources but also keeps track of class notes effectively.

Our application uniquely allows users to supplement their knowledge with YouTube videos, utilizing both Symbl.AI and YouTube to enhance the educational experience.

Built With

flask, sqlite, symbl.ai
